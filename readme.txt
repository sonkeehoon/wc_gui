
## 개인 프로젝트 : 단어만 입력하면! 네이버 검색 결과를 가져오고 
워드 클라우드까지 만들어주는 gui 프로그램
작업 기간 : 
2022.11.30 ~ 2022.12.01,
2022.12.09 ~ 2022.12.10

컴퓨터에 파이썬이 설치되어 있다고 가정한다.
**중요
우선 필요한 파이썬 패키지를 설치해야 한다.
패키지 한꺼번에 설치 : pip install -r requirements.txt
모든 패키지를 설치한 후 아래 순서대로 진행

1. gui_wc.py를 실행
2. '검색어 입력' 란에 원하는 검색어를 입력 한다. (아직 '만들기' 버튼은 누르면 안된다!)
3. '저장경로' 란에서 '찾아보기' 버튼을 클릭후 저장하고 싶은 경로를 지정한다.
4. 저장경로까지 지정한 후 '만들기' 버튼을 누르고 몇초간 기다린다.
5. 지정한 경로에 워드 클라우드 이미지가 생성된다.
